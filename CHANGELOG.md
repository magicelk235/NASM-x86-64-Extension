# Changelog

## [1.0.0] - 2026-05-16

### Added
- Semantic highlighting for instructions, registers, labels, and preprocessor directives
- Project-wide symbol discovery across open files
- Local label intelligence with `.` prefix support
- Numeric hover with decimal, hex, and binary conversions
- IEEE 754 float64 bit representation on hover
- Knowledge base with 684 x86-64 instructions and NASM directives
- Support for `%macro`, `%define`, `%assign`, `%iassign` preprocessor directives
- Auto-completion for instructions, registers, and user-defined symbols
